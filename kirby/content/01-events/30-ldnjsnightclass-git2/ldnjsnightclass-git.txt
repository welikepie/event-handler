### Who is it for?
This workshop is for those who want to learn all the things they should have learned when they started out with Git. Go from being a complete beginner to highly proficient; learning the theory behind Git, all the necessary commands you need to get by and more advanced things like publishing patches and creating Git hooks. 

### Who is taught by?

This workshop is being taught by Kornel Lesinski (link: https://twitter.com/pornelski text: @pornelski), Senior Developer at BazaarVoice. Kornel is a developer with many languages in his toolset including PHP, HTML5, CSS & JavaScript. He has created many popular image compression tools: (link: http://www.imageoptim.com text: ImageOptim), (link: http://www.pngmini.com text: ImageAlpha) & (link: http://www.pngquant.org text: pngquant) and is a big fan of web-based game development and web standards. You'll find Kornel talking at meet-ups, events and international conferences. 

### What you'll learn

* The theory of Git: How it works and basic terms & concepts.
* How to commit, push, pull and move files. 
* How to undo mistakes
* How to recover from the “strange and stupid” situations that arise when using Git
* How to accept and publish patches
* How to deploy your changes to your live server. 
* How to create Git Hooks; where Git will perform an action of your choice upon receiving a new commit (send an email notification of changes, deploy to website)
