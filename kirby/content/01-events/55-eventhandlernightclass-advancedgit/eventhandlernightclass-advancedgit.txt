On Tuesday the 18th of February, (link: http://www.twitter.com/pornelski text: Kornel Lesiński) will be holding an evening workshop on how to get more out of Git and become a power user.

(image: gitworkshop.png class: full-width)

### Who is it for?
This workshop is for those who already know how to use Git, but would like to become a power user. Learn how to manipulate branches and history to any shape you like, get more information from git and learn how to fix "messed" up repositories, as well as learning to automate Git.  

### Who is taught by?

This workshop is being taught by Kornel Lesinski (link: http://twitter.com/pornelski text: @pornelski), Senior Developer at BazaarVoice. Kornel is a developer with many languages in his toolset including PHP, HTML5, CSS & JavaScript. He has created many popular image compression tools: (link: http://www.imageoptim.com text: ImageOptim), (link: http://www.pngmini.com text: ImageAlpha) & (link: http://www.pngquant.org text: pngquant) and is a big fan of web-based game development and web standards. You'll find Kornel talking at meet-ups, events and international conferences. 

### What you'll learn

* An Overview of git's architecture that explains git's strengths and quirks, e.g. why hashes? What is rebase _really_ doing?
* Importing code, history and branches from other systems like SVN, CVS, Bazaar.
* How to recover from screwed up rebases, messed up branches.
* Power-user options for resolving merge/rebase conflicts.
* Splitting, filtering and joining git repositories.
* Automating actions when commits or pushes are made:
- Enforcing coding standards, running unit tests
- Deploying code and tagging deployed versions
** Hidden gems and trips & tricks.

### Preparation

* An understanding of and ability to use the basics of Git
* Please bring your laptop
