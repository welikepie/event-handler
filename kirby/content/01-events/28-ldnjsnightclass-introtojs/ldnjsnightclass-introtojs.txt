## Who's it for?

Have you been meaning to learn JavaScript but have had difficulty setting aside time to do so or just don't know where to start? This workshop is for people who are interested in learning JavaScript but have no/little previous experience with it. 

In this course, for a few hours each week, you will be given a hands on approach to JavaScript with Daniel taking you through each core aspect of the language. 

If you have any questions about the course, get in touch at (email: contact@eventhandler.co.uk text: contact@eventhandler.co.uk).

## Who's teaching?

This workshop is being run by Daniel Knell (link: http://www.twitter.com/danielknell text: Daniel Knell ), a Senior Developer proficient in several languages, of which JavaScript is just one. Daniel has created many interesting things at a number of hackdays, winning both (link: http://envhack.com text: env[:hack]) and (link: http://rewiredstate.org/hacks/national-hack-the-government-2013 text: National Hack the Government Day). He also speaks regularly at a number of meet-ups across London. 

## What you will learn:

You will cover the fundamentals of the JavaScript programming language, starting with the basics and moving on to more advanced concepts with an end goal of creating a simple game purely in JavaScript.

* Basic Data Types (Strings, Numbers, etc.)
* Maths in JavaScript
* Classes
* Objects
* Arrays
* Functions
* Basic DOM (Document Object Model) operations
* How to build a simple game solely with JavaScript
