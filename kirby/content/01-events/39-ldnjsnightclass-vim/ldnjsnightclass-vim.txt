(image: vimworkshop.png class: full-width)

### Who is it for?

Have you been curious about the Vim editor, but didn't know where to start? Worried about losing days while you try to figure out how to do the basics? Join Alex Pounds for this beginner's workshop. You'll learn how to use Vim & how to get the best out of it as a programmer's editor, as well as how to think like a Vim user & work effectively. You'll also meet some of its more advanced features, and take a look at some of the more popular plugins.
 

### Who is it taught by?

This workshop is being run by (link: http://www.twitter.com/acreature text: Alex Pounds), who has been an avid Vim user for over a decade. Alex is a freelance web developer and Last.fm alumnus, working in London. When not creating things with PHP, Python, JavaScript or Ruby; Alex is a keen photographer and tea-drinker. 

### What you'll learn

* An introduction to Vim and why you should use it
* The Vim mental model - thinking like a Vim user
* The basics of Vim
* Working with code: syntax highlighting, autocompletion, running tests and more
* Advanced editing: Working with registers, macros, external commands and more
* Extensions and Package Management

### Requirements

* Please make sure you bring your laptop
