On the 25th of February, (link: http://www.twitter.com/timruffles text: Tim Ruffles) will be hosting a workshop in which you can learn to test JavaScript. 

(image: testingjs.png class: full-width)

### Who is it for?

Testing lets you ship better code with less doubt. It's a fantastic feeling to have a good level of testing in your team's code. This class will give you all the tools you need to create and run a test suite on Javascript projects in the browser, or in node.js.

### Who is it taught by?

This workshop is being run by (link: http://www.twitter.com/timruffles text: Tim Ruffles). Tim is a Ruby & JavaScript Developer living and working in London. Having previously worked for Skimlinks as their tech lead, he now spends time mentoring others and working on (link: http://www.sidekickjs.com text: SidekickJS), his code-quality tracking tool. You can find him speaking about the web at (link: http://www.lanyrd.com/timruffles text: various events). 

### What you'll learn

* Your first test
* Core idea of testing
* Assertion technique
* Setup/tear down
* Contexts and test groups
* Testing DOM
* Testing async
* Testing OO code + Javascript's testing secret
* Mocking
* Stubbing
* Stubbing an API
* Integration testing
* Continuous integration with Jenkins
* Testing on many browsers with Karma

### Requirements

* Must have a competency with JavaScript
* Please bring your laptop with your text editor of choice installed
