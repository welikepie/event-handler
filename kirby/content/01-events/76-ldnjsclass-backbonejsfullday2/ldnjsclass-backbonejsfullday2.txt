Title: London JS Class: BackboneJS Full-Day Workshop
----
Text: 
On the 21st of March, (link: http://www.twitter.com/timruffles text: Tim Ruffles) will be hosting a **full-day** workshop in which you can learn to use Backbone.js to create web applications. 

(image: backbone.png class: full-width)

### Who is it for?

This workshop is for those who want to learn how to build scalable JavaScript applications.

Discover how to work with Backbone.js to create testable, maintable code with a clear structure. This class will take your existing JS, HTML and CSS skills and allow you to create modern thick-client applications using Backbone. 
 

### Who is it taught by?

This workshop is being run by (link: http://www.twitter.com/timruffles text: Tim Ruffles). Tim is a Ruby & JavaScript Developer living and working in London. Having previously worked for Skimlinks as their tech lead, he now spends time mentoring others and working on (link: http://www.sidekickjs.com text: SidekickJS), his code-quality tracking tool. You can find him speaking about Sidekick and other aspects of the web at (link: http://www.lanyrd.com/timruffles text: various events). 

### What you'll learn

* Getting started with Backbone.js
* Models & Collections
* Views
* Routers
* Application structure
* TDD - Test-Driven Development
* Plugins, templates and extensions
* Extending Backbone.sync

### Requirements

* Please bring your laptop
* To take part in this course, you must be well-versed in JavaScript, jQuery (or similar DOM library) and HTML/CSS.
